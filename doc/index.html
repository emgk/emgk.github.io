Guten Block Fields

Guten Block Fields is NPM package which helps you to create inspect control fields for your Gutenberg's block.

Installation

Before you install this package make sure Node.js is installed on your system. If not, you can get it from here http://nodejs.org/en/download

Install this package by using this command:

npm install guten-block-fields

Commands

  • mkdocs make - Generate the fields.
  • mkdocs version - Get the current version of the plugin.
  • mkdocs help - Print this help message.
